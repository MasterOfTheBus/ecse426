Configure a GPIO port and Initialize it.

The first two parameters correspond to selecting the specific GPIO port and bus. The remaining parameters are used to set the values of the GPIO initilization struct (GPIO_InitTypeDef).

Parameters
GPIOxSpecifies the GPIO port to enable
periph_GPIOxSpecifies the GPIO port to enable for the bus clock.
pinsSpecifies the GPIO pins to be configured.
modeSpecifies the operating mode for the selected pins.
speedSpecifies the speed for the selected pins.
oTypeSpecifies the operating output type for the selected pins.
puPdSpecifies the operating Pull-up/Pull down for the selected pins.
Return values
none
void DisplaySingleLED ( int  LED_num)

Select one of the four LEDs to turn on.

The LEDs are hardwired to GPIO port D. The modulo of LED_num is taken to determine which LED to light up.

The modulo values correspond accordingly:

  • 0: GPIO pin 12
  • 1: GPIO pin 13
  • 2: GPIO pin 14

  • 3: GPIO pin 15

    Parameters
    LED_numA number selecting the LED to display.
    Return values
    None.
