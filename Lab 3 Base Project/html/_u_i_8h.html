void Keypad_read ( void  )

Assembling user input.

  • Looking for user input
  • Update when change detected
  • Handle keypad bounce problem by counting the number if consecutive idle
  • Exit when ENTER is pressed
  • Special mappings interpretation: A ---— value = 10 — Equivalent to HEX representation (not used) B ---— value = 11 — Equivalent to HEX representation (not used) C ---— value = 12 — Equivalent to HEX representation (not used) D ---— value = 13 — Equivalent to HEX representation (not used)

    ---— value = 22 — Used as ENTER — Exit the function

  • ---— value = 21 — Used for calibration (not used) idle — value = 99 — Result when nothing is pressed — Continue waiting for input
